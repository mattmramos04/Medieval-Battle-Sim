# Medieval-Battle-Sim
This is a simple printing game that shows off a simulation of an ancient battle between Rome and Carthage using dicts and functions and core python concepts as a first real python project. 
## Concepts Used:
Nested dicts
if loops
functions
conditionals
f-strings

## Features:
Dynamic Battle System with Officer Buffs
Morale System
Historical Theming
Narration

This is real barebones. It is my first project and so I intend to make better projects, hence it is literally just a simple run loop.
